Date Functions

Single-row date functions that return values of datatype DATE, except MONTHS_BETWEEN, which returns a number.

Function

Value Returned

ADD_MONTHS(d,n)

Date d plus n months

LAST_DAY(d)

Date of last day of month containing d

MONTHS_BETWEEN(d,e)

Number of months by which e precedes d

NEW_TIME(d,a,b)

Date and time in time zone b when date and time in zone a are d. a and b are CHAR values identifying time zones.

NEXT_DAY(d,char)

Date of the first day of the week named by char that is equal to or later than the date d

SYSDATE

Current date and time

Date Truncation and Rounding

Function

Value Returned

ROUND(d,fmt)

d rounded as specified by rounding unit fmt

TRUNC(d,fmt)
